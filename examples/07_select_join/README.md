# Cake demo app: SELECT with JOIN

This demo of Cake shows how to `SELECT` with a `JOIN`.

To start postgres via docker compose, see
[examples/README.md](../../README.md#Installing-prerequisites), then:

```shell
gleam clean
gleam run
```
