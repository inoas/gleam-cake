# Cake demo app: INSERT

This demo of Cake shows how to `INSERT ON CONFLICT UPDATE`.

To start postgres via docker compose, see
[docs/demo-apps/README.md](../../README.md#Installing-prerequisites), then:

```shell
gleam clean
gleam run
```
