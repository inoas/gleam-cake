# Cake demo app: SELECT and decode

This demo of Cake shows how to `SELECT` and decode returning rows into
Gleam records.

To start postgres via docker compose, see [/docs/demo-apps.md](../../demo-apps.md#Installing-prerequisites)

```shell
gleam run
```
